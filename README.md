# 我的餐廳清單

使用Node.js, Express, Express-handlebars 套件製作而成

## 功能清單
* 點選每一家餐廳都有詳細餐廳詳細的說明
* 點選"我的餐廳清單"即可重新整理
* 可以透過搜尋店家名稱，找到自己要的餐廳

## 環境需求
* Node.js: v10.15.0
* express: v4.17.1
* express-handlebars: v5.1.0

## 啟動方式
* 將專案下載至本機內

  ``git clone https://github.com/hsiyu1121/restaurant_list.git``
* 切換至資料夾內

  ``cd restaurant_list``
* 安裝相關的套件

  ``npm install``
* 執行程式

  ``npm run start``
* 開啟瀏覽器輸入以下網址

  ``http://localhost:3000``
